# DebugMaster

A professional debugging platform designed for developer drills.

## Changes Made

- Updated typography to Inter for high-readability technical UI.
- Refined design tokens with sharper borders and subtle shadows for a premium feel.
- Refactored the Sandbox into a multi-pane IDE layout featuring a Challenge Navigator, Code Editor, and integrated Results/Preview panel.
- Redesigned the Home page with a focused hero section and performance statistics.
- Converted the Problem Set and Submission Ledger into clean, information-dense tables.
- Standardized button and component styles across all views for consistency.

## Tech Stack

- React 19
- Tailwind CSS
- Monaco Editor
- Lucide React

Developed with a focus on speed, precision, and senior-level debugging instincts.
